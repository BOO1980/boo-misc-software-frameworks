Deploy Bot Overview

  • Deploy bot deploys code from a hosted Git repostitry to anywhere you need it shipped
  • It allows a single developer or an entire development team to follow a consistant process for shipping code
  • There is no need to manually track changes or log onto a server to upload files any longer
  • With deploybot you can review files set to deploy before you start the process
  • Take your deployment process to the next level by using a build server with a predefined docker container or a custom container
  • During deployment in the container fetch dependancies like NPM or composer
  • It compiles code and build compress files with tools like grunt or gulp
